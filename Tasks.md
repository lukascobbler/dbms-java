## 11 JDBC

Although a lot of the features of the system are JDBC compliant, I decided to skip implementing the JDBC interface, and instead opted in for the raw server-client query console approach where query plans are executed immediately and tables are printed. Even though, the JDBC chapter defines some universal tasks like graceful shutdown and manual commit, which are implemented.
